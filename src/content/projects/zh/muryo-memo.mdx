Muryo Memo 源于我自己作为同人展参与者的真实需求。每次参加漫展或粉丝聚会，我都会自己制作周边小物和大家交换，但久而久之发现没有一个顺手的地方来记录：已经做了什么、还需要准备什么、要和哪些人交换。电脑文件夹不能随身带，备忘录太散，Notion 又不是我的首选。所以我想做一个专门为这件事设计的小工具，一个只做这一件事的备忘录。  

技术上，项目采用 React 前端搭配 Express + MongoDB 后端，实现了基础的产品管理 CRUD API。页面背景使用 p5.js 与 matter-js 构建了一套物理驱动的动态粒子效果，sketch 以 instance mode 运行并封装为独立模块，通过 React ColorContext 与界面组件共享调色板状态。整个项目部署在 Digital Ocean 上。
