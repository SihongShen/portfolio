我负责了这个 Web 应用的前端开发，主要工作集中在三个方向：实现推荐面板的视觉布局系统、构建用户兴趣档案的数据可视化组件，以及完成整体的组件库搭建与状态管理。项目基于 Next.js + React，使用 TailwindCSS 处理样式，并与后端 FastAPI 服务通信。  

技术细节上，推荐面板的卡片布局采用了 BSP（二分空间分割）算法，在严格的三列、最多三行的约束下递归切分空间，配合宽高比评分函数优先选择接近正方形的分割方案，并引入预设列宽变体实现视觉上的不规则感。数据可视化部分用 Chart.js 实现了六边形雷达图（展示用户兴趣的六大类别分布）和水平条形图（展示前十五个子类别标签频率）。每张 Salon 推荐卡片的字重、字号、对齐方式均通过基于卡片 ID 的伪随机函数确定，保证每次渲染结果一致但视觉上富有变化。同时增加了hover等图标交互细节。全局状态通过 AuthContext 和 AppStateContext 两个 Context 管理认证信息与应用数据。
