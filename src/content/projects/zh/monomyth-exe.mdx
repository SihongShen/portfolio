MONOMYTH.exe 是一个以约瑟夫·坎贝尔"英雄之旅"理论为骨架的实验性叙事引擎，也是我上一个项目 [Before the Trial](/zh/projects/before-the-trial) 的延伸探索。我想做一个真正无限的文字冒险游戏——每次体验都由用户输入的一个词语作为种子生成，故事沿着六个神话阶段展开分支。六这个数字是经过测试调整的结果：五个阶段让玩家觉得故事没讲完，六个恰好给人完整感。除了文字，我还想让整个体验有更强的仪式感：背景会根据故事当前状态实时生成对应的图标，用手势就能做出选择，故事结束后可以下载一份带有专属封面的 PDF 存档。  

技术上，项目用 React + Vite 构建，界面设计为复古终端风格，带有 CRT 扫描线和打字机效果。叙事核心由 Google Gemini API 驱动——经过大量 prompt 调试：强制第三人称叙述、锁定古希腊神话风格、规范 JSON 输出格式、以及让选项更开放而非简单的是/否题。背景图标通过 Gemini 图像生成实时产出，封面同理。手势交互部分使用 MediaPipe 预训练手部模型，通过指尖位置判断用户的选择，并以 p5.js 绘制关键点做视觉反馈。
