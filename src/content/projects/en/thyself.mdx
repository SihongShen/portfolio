I worked on the frontend of this web application, with work across three main areas: building the visual layout system for the recommendation panel, developing data visualization components for user interest profiles, and assembling the shared component library alongside state management. The project is built with Next.js and React, styled with TailwindCSS, and communicates with a FastAPI backend.  

On the technical side, the recommendation panel uses a BSP (Binary Space Partition) algorithm to recursively divide the layout space under strict constraints — exactly three columns and at most three rows — selecting split ratios based on an aspect ratio scoring function that favors layouts closer to square. A set of predefined column-width variations introduces visual irregularity while keeping the structure consistent. For data visualization, I implemented a hexagonal radar chart in Chart.js to display the distribution across six top-level interest categories, alongside a horizontal bar chart ranking the top fifteen sub-category tags by frequency. Individual Salon card styles — font weight, size, and alignment — are determined by a seeded pseudo-random function keyed to each card's ID, ensuring consistent but visually varied rendering across sessions. Interaction details such as hover states and icon animations were also added throughout. Global state is managed through two React Contexts: AuthContext for authentication and AppStateContext for application-level data.
