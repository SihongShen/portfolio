MONOMYTH.exe is an experimental narrative engine built around Joseph Campbell's Hero's Journey — and a natural extension of my earlier project [Before the Trial](/en/projects/before-the-trial). The idea was to create a genuinely infinite text-adventure experience, where a single word typed by the user seeds an entire branching myth across six narrative stages. That number wasn't arbitrary: testing with five stages left players feeling the story was unfinished, while six gave a satisfying sense of completion. Beyond the text itself, I wanted the experience to carry a certain ritual weight — the background generates iconography in real time based on the story's current state, choices can be made through hand gestures, and finishing a journey produces a downloadable PDF log with a uniquely generated cover.  

The project is built with React and Vite, styled as a retro terminal interface complete with CRT scanlines and typewriter effects. The narrative core runs on the Google Gemini API, shaped through extensive prompt engineering: enforcing third-person narration, locking the tone to an ancient Greek mythological register, structuring outputs as JSON, and rewriting choice prompts to avoid the yes/no patterns that LLMs tend to default to. Background icons and story covers are generated via Gemini's image API. The gesture interaction layer uses MediaPipe's pretrained hand model to read fingertip position as a choice input, with p5.js rendering the hand keypoints as visual feedback.
